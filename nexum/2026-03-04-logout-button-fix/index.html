Corretto il pulsante di uscita

Il pulsante per uscire dal sistema (in alto a destra nella barra) aveva il testo dello stesso colore dello sfondo, rendendolo praticamente invisibile.

Abbiamo corretto il colore: ora l'icona di uscita appare bianca su sfondo scuro, ben visibile e cliccabile.

La barra superiore dopo la correzione: a destra si vede l'email, lo store e il pulsante di uscita.
Riepilogo tecnico

Server: Cloudways Athena (staging)

File: resources/views/components/layouts/head-style.blade.php

Problema: Il selettore CSS per il bottone logout era sbagliato. Usava #app #app-body #topbar ... ma nella struttura HTML, #app-body (il tag body) contiene #app (un div), non il contrario. Il selettore non matchava mai.

Verifica Playwright:

btn-logout color: rgb(255, 255, 255) ← bianco btn-logout bg: rgb(85, 85, 85) ← grigio scuro topbar bg: rgb(238, 238, 238) ← grigio chiaro

Fix applicato:

/* PRIMA (selettore sbagliato — non matchava mai) */ #app #app-body #topbar #topbar-right #btn-logout { color: #fff; } /* DOPO (selettore corretto) */ #topbar-right #btn-logout { color: #fff; }
